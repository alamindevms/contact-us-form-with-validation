# Contact us form with validation

A robust Contact Us form with Name, Email, Phone, Subject, and Message fields. User-friendly validation and error handling, and reset form inputs after submission for a seamless experience.

## Setup

Node version:

```
use: v18
```

Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev
```

## Test ( Unit, E2E )

Test the form inputs:

```bash
# npm
npm run test

# yarn
yarn test

```

Test the form e2e:

```bash
# npm
npm run test:e2e

# yarn
yarn test:e2e

```

## Production

Build the application for production:

```bash
# npm
npm run build

# pnpm
pnpm run build

# yarn
yarn build
```

Locally preview production build:

```bash
# npm
npm run preview

# pnpm
pnpm run preview

# yarn
yarn preview
```
